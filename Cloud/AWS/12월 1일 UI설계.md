### UI설계

1. 메인페이지(달력 형식 아주 좋음!!) - 사실 없던 기능
   - 왼쪽 선택란 제외하고 슬라이드 형식 지원 -> **O**
   - 간단한 제목 정도 함께 표시하면 좋을 듯함 -> 깜빡하고 예기안함
2. 내 옷장 - 자신이 가지고 있는 옷
   - 캘린더에 저장 기능 좋음 => **O**
   - 지정한 의류에 맞는 코디 추천기능 추가 => **X**
3. 유행도 평가
   - 현재 코디 이미지를 저장 후 인식하여 유행도 평가 => **O**
   - 여기에 지정한 의류에 맞는 코디 기능을 추가할 것인가? => **X**
4. 코디 추천
   - 부분적으로 추천기능 추가! => **보류**
5. 일정 저장
   - 내옷장과 일정 클릭의 기능이 유사함!! -> 하나의 페이지로 구성 => **O**
   - 내 옷장과 일정클릭시 나오는 페이지를 동일하게 구성 
      -차이점: 만족도 조사 -> 어디다 활용할 것인가 => **후순위 추가**
