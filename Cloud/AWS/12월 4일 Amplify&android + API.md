## API (Application Programming Interface)

UI 가 사용자와 사용자가 다룰 대상(H/W or S/W)을 연결

API 는 프로그램과 또 다른 프로그램을 연결해주는 다리 역할



### Amplify 객체 저장소 스토리지 s3 활성화
